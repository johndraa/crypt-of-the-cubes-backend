This repository contains the backend services for **Crypt of the Cubes**, a multiplayer roguelike game project developed as part of a university software engineering course.

The backend provides REST APIs and persistent storage for player accounts, character progression, and match data, supporting multiplayer gameplay features.

## Tech Stack
- Java
- Spring Boot
- Maven
- MySQL
- JPA / Hibernate
- REST APIs
- Linux deployment environment

## Features
- Player account management
- Character and progression persistence
- Match and participant data handling
- Database-backed game state storage
- REST endpoints for client-server communication

## Notes
This repository represents a completed academic project and is maintained as a portfolio example.
